Loading and Storing Assets

To handle import and export of meshes or image files - Assets - CrossForge implements the class SAssetIO. To load an Asset, simply use the load function in combination with the class T3DMesh or T2DImage depending on the asset to store it:

T3DMesh<float> M;
T2DImage<uint8_t> I;
SAssetIO::load("Assets/path/mesh.meshformat", &M);
SAssetIO::load("Assets/path/image.imageformat", &I);
